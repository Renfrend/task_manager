<h1>Task manager</h1>

<h2>Install dependencies for the frontend</h2>

```sh
cd frontend
npm install
```

<h2>Install dependencies for the backend</h2>

```sh
cd backend
npm install
```

<h2>Run backend</h2>

```sh
cd backend
npm run start:dev
```

<h2>Run frontend</h2>

```sh
cd frontend
npm start
```
